Projeto em Desenvolvimento - Sistema de Cadastro de Clientes (C#)

Sistema desktop desenvolvido em C# com foco no aprendizado prático de desenvolvimento de sistemas e regras de negócio. O projeto contempla cadastro, atualização, consulta e exclusão de clientes, utilizando uma estrutura organizada, lógica orientada a objetos e boas práticas de código. O objetivo é simular um sistema real utilizado em ambientes corporativos, evoluindo continuamente conforme novos conhecimentos são adquiridos.

Projeto em Planejamento – Análise de Clientes e Indicadores

Projeto voltado à análise de dados de clientes com foco na geração de indicadores e apoio à tomada de decisão. Este projeto reflete meu interesse pela área de dados e pela conexão entre tecnologia e decisões estratégicas, buscando aplicar análise de dados de forma prática e orientada a resultados.
